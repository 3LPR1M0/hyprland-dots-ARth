# hyprland-dots-ARth

Hyprland.conf file is devided the physical hyprland.conf file remains in its standard location in ~/.config/hypr/hyprland.conf. However everything inside it has beeen subdivided in a config folder and its all in different files under the ~/.config/hypr/configs directory you dont have to do this if you dont want to I just think it's easier to navigate with yazi. 
This is a very basic and bare bones hyprland config however its decent if you're starting out but there's a lot more you could do as you go through the files you'll see a lot of it is unedited or barely.
