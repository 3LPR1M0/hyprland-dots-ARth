# hyprland-dots-ARth

Hyprland.conf file is devided the physical hyprland.conf file remains in its standard location in ~/.config/hypr/hyprland.conf. However everything inside it has beeen subdivided in a config folder and its all in different files under the ~/.config/hypr/configs directory you dont have to do this if you dont want to I just think it's easier to navigate with yazi. 
This is a very basic and bare bones hyprland config however its decent if you're starting out but there's a lot more you could do as you go through the files you'll see a lot of it is unedited or barely.


As for the Waybar I would recommend you to try waybar and see its limitations and advantages. I think the Waybar is great for new Hyprland or Niri users, like extremely minimal set ups or like a I3 type of rice. However as you progress in the ricing jurney things like quick shell, eww, ad noctalia shell would be things to look into. 


I was using wallust in this hyprland set up so you can see the @color12 etc in the style.css file and also the import line at the very top. I wouldn't recommend using wallust I think manually making themes and using a different approach to themeing is better. However if you do want to try it by all means it's a great starting point setting up your wallust config.


I would recommend not to just blindly copy and paste from the files you should use this as an example as to what you could do. Anyways enjoy!!
